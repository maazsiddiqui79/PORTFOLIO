TITLE:
Blog Website	

DESC:
A blog website where users can create an account, write and post blogs, comment on posts, and view blogs by others.	

BODY:
<h1>Blog Web Application</h1> <p><a href="https://github.com/maazsiddiqui79/Blog-Website" target="_blank">GitHub Repository</a> | <a href="/" target="_blank">Live Website</a></p> <hr /> <h2><strong>Type:</strong> Web</h2> <hr /> <h2><strong>Short Description</strong></h2> <p>A secure and responsive blog platform where users can register, log in, and publish content-rich articles. Includes hashed password authentication, role-based access, and a WYSIWYG editor for rich formatting.</p> <h2>User Authentication (Hashed Passwords)</h2> <p>Users must sign up with a valid email and password. Passwords are hashed using bcrypt or SHA-256 and stored securely. Sessions are protected with secure cookies and CSRF tokens for safe form handling.</p> <h2>Create and Manage Blog Posts</h2> <p>Authenticated users can write, edit, and delete their posts using a CKEditor-powered rich-text interface. Tags, categories, and featured images are supported for better content organization.</p> <h2>Read-Only View for Guests</h2> <p>Visitors can browse blog posts without logging in. Only registered users can comment or create new posts. Admins can manage all content and users.</p> <h2>Example Use Case</h2> <p>Ideal for personal blogging, academic journals, or team announcements. Supports both single-author and multi-author workflows with clear access controls.</p> <h2>Security Features</h2> <ul> <li>Secure authentication with hashed passwords</li> <li>Session management using HTTP-only cookies</li> <li>CSRF protection on all forms</li> <li>Input validation and sanitization</li> </ul> <h2>Key Features</h2> <ul> <li>Create, update, and delete blog posts</li> <li>Rich text editing with embedded images and links</li> <li>Comment system with moderation tools</li> <li>Search and category filtering</li> <li>Responsive UI with clean design</li> </ul> <hr />
